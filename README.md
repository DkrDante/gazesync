GazeSync

Python
📝 Description

Gazesync is a high-performance Python framework designed to bridge the gap between human visual attention and digital environments. By leveraging advanced computer vision and real-time data processing, Gazesync allows developers to seamlessly synchronize gaze-tracking data with interactive elements, media playback, or collaborative workflows. Whether it is for improving accessibility, conducting deep-dive user research, or creating immersive gaze-contingent displays, Gazesync provides an intuitive and extensible toolset for building the next generation of eye-aware software applications.

🛠️ Development Setup
Python Setup

    Install Python (v3.8+ recommended)
    Create a virtual environment: python -m venv venv
    Activate the environment:
        Windows: venv\Scripts\activate
        Unix/MacOS: source venv/bin/activate
    Install dependencies: pip install -r requirements.txt
